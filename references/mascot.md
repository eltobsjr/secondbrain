# SecondBrain — Mascote

Ao iniciar qualquer skill SecondBrain, imprima **exatamente** o seguinte bloco como
primeira coisa na resposta, antes de qualquer texto:

```
  ██████████████████████████
  █                        █
  █   .-"""""-..-""""-.    █
  █  /  .-. .-Y-. .-. \   █
  █ |  / )  |   |  ( \  |  █
  █ | |  \__|___|__/  | |  █
  █ |  \ (  ~~~~~ ) /  |  █
  █  \  `-.______.-'  /   █
  █   `-.___________.-'   █
  █                        █
  █     S E C O N D        █
  █     B R A I N          █
  ██████████████████████████
```

Substitua a última linha do bloco interior pelo nome do comando sendo executado:

| Skill | Linha de comando |
|---|---|
| setup | `  /secondbrain-setup  ` |
| devtrack | `  /devtrack            ` |
| resume | `  /resume              ` |
| context | `  /context             ` |
| learn | `  /learn               ` |
| decision | `  /decision            ` |
| priority | `  /priority            ` |
| feature | `  /feature             ` |
| audit | `  /audit               ` |
| readme | `  /readme              ` |
| standup | `  /standup             ` |
| changelog | `  /changelog           ` |
| onboarding | `  /onboarding          ` |
| postmortem | `  /postmortem          ` |
| rfc | `  /rfc                 ` |
| release | `  /release             ` |
